Why Building AWS Lambda Using Springboot¶

다양한 문제들의 맥락에서 AWS Lambda는 솔루션이 될 수 있다. 사용량 기반의 서버 어플리케이션 호스팅이 필요한 경우가 대표적이다. 실질적인 비즈니스 모델이 명확하지 않은 시스템을 구현하고 동작 시켜야 할 때, 일정하지 않은 요청을 처리하기 위해서 자원을 계속 운영하는 것은 부담이기 때문이다. 문제는 생산성이다. 팀의 구성원들의 숙련도가 높은 기술 스택이 높은 생산성으로 이어지는 경우가 많다. Springboot 프레임워크의 생산성은 각자의 경험과 함께 공개된 수많은 시행 착오와 재사용 가능한 코드 조각들을 고려해서 생각해야 한다.
